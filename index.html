SixthSense

A software interface to the real world

What is SixthSense?

SixthSense is an open source project that uses a projector and camera to create finger-based interaction applications (check out the video for more).

The code is written in C# with .NET, and currently does not run on Mono. However, there are several ports in progress to Android, iOS, Linux and more.
